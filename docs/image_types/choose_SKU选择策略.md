# 03 Choose｜SKU 图

## Purpose

准确呈现颜色、尺寸、型号和套餐差异，帮助用户完成选择。

## User Psychology

选择困难来自信息不完整、命名混乱和差异不可见。用户需要确认“有哪些选项、它们如何不同、我该选哪个”。

## Good Examples

- SKU 与官方清单一一对应，顺序稳定。
- 色卡、型号或尺寸使用统一比例和视角。
- 背景中性，不改变产品辨色。

## Bad Examples

- AI 生成不存在的 SKU 或错误颜色。
- 标签与产品错位，套餐数量不一致。
- 强色偏灯光导致用户误判颜色。

## Optimization Notes

- 先保证数据准确，再优化排列美观。
- SKU 较多时按系列、色系或用途分组，避免一次塞满。

## Future Ideas

- 为不同品类定义 SKU 分组规则。
- 建立色差校验与屏幕显示风险说明。
