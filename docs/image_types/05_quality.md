# 05 Quality｜细节图

## Purpose

通过材质、工艺和结构细节证明产品品质。

## User Psychology

当用户无法触摸产品时，会通过纹理、反射、连接处和结构判断做工是否可靠。

## Good Examples

- 微距展示刷头、瓶口、金属连接或表面纹理。
- 玻璃、磨砂、珠光等材质反馈真实。
- 高光和阴影能解释结构，而不是掩盖缺陷。

## Bad Examples

- AI 添加不存在的纹理、部件或材质。
- 过度锐化、塑料感、错误反射。
- 特写区域与实际产品结构不一致。

## Optimization Notes

- 每张细节图只证明一到两个关键品质点。
- 对照高清实物素材检查边缘、接口和反射。

## Future Ideas

- 建立品类关键细节清单。
- 为高风险结构增加局部真实性审核标准。
