# 06 Use｜使用图

## Purpose

展示正确、自然的使用过程，证明产品易用且适合目标场景。

## User Psychology

用户希望降低学习成本和使用风险。正确动作能回答“怎么用、是否顺手、是否适合我”。

## Good Examples

- 手势、握持、方向和产品比例符合现实。
- 场景与目标用户、产品用途一致。
- 产品在动作中仍清晰可见。

## Bad Examples

- 危险操作、错误握持、悬浮产品或畸形手部。
- 擅自增加配件，改变产品尺寸。
- 场景漂亮但无法说明产品如何使用。

## Optimization Notes

- 先用真实操作流程定义关键动作，再设计画面。
- 易用性应通过动作和结果表达，而不是只写“Easy to Use”。

## Future Ideas

- 为不同品类建立标准动作参考。
- 增加安全使用和常见错误示例库。
