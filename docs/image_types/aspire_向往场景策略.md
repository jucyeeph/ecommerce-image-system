# 07 Aspire｜向往图

## Purpose

把产品放入相关的理想环境，帮助用户投射身份与生活方式。

## User Psychology

用户会通过场景判断产品是否属于自己想成为的人、想进入的环境或想拥有的生活。

## Good Examples

- 美甲产品进入专业工作室，宠物用品进入高级但真实的护理场景。
- 环境强化品牌价值，产品仍可识别。
- 人物、空间和文化语境符合目标市场。

## Bad Examples

- 无关豪车、珠宝或奢侈品堆砌。
- 复制竞品门店、商标或可识别视觉资产。
- 环境成为唯一主角，产品几乎消失。

## Optimization Notes

- 场景必须回答“用户向往什么”，不能只回答“什么看起来贵”。
- 记录目标市场对专业、家庭、礼赠或社交身份的不同偏好。

## Future Ideas

- 建立场景与用户画像映射库。
- 按市场测试专业身份与生活方式身份的转化差异。
