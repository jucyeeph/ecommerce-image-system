# 08 Premium｜价值图

## Purpose

通过真实包装、套装、收纳和开箱体验提升感知价值。

## User Psychology

完整、有秩序的交付体验会强化“值得购买、适合送礼、品牌可靠”的判断。

## Good Examples

- 展示真实存在的礼盒、包装、配件和数量。
- 陈列整齐，材质和开箱顺序清楚。
- 画面体现完整体验，不依赖夸张装饰。

## Bad Examples

- 虚构礼盒、赠品、配件或包装版本。
- 改变包装文字、Logo 和产品数量。
- 用无关奢侈品道具制造虚假价值。

## Optimization Notes

- 价值感首先来自真实交付的完整度，其次才是摄影表现。
- 套装图必须与 SKU 和仓储交付清单同步。

## Future Ideas

- 建立礼赠、专业套装和入门套装的视觉模式。
- 记录开箱内容对客单价和套装转化的影响。
