# 04 Desire｜效果图

## Purpose

展示产品带来的可信结果，让用户想象拥有或使用后的状态。

## User Psychology

用户不只购买产品本身，也购买预期结果。效果越具体、真实、可代入，购买欲越强。

## Good Examples

- 甲油胶展示真实上手效果，工具展示可证明的处理结果。
- 使用对象、比例、颜色和材质符合现实。
- 效果明显但不过度夸张。

## Bad Examples

- 伪造使用前后差异或暗示未经批准的功效。
- 出现畸形手部、错误使用部位或不可能的结果。
- 只依赖豪华场景，没有展示产品结果。

## Optimization Notes

- 明确效果证据来自实拍、产品测试还是经批准的示意。
- 优化重点是代入感和可信度，而不是夸张程度。

## Future Ideas

- 建立不同品类的结果表现边界。
- 记录效果图对收藏、加购和转化的影响。
