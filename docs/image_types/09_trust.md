# 09 Trust｜信任图

## Purpose

用可核验的品质、服务和保障信息消除购买前的最后顾虑。

## User Psychology

临近下单时，用户更关注风险：是否真实、是否安全、出了问题怎么办。清楚且有依据的承诺能降低风险感。

## Good Examples

- 认证、检测、售后和保障均有明确来源。
- 信息克制、正式、适合移动端阅读。
- 区分品牌承诺、平台政策和第三方认证。

## Bad Examples

- AI 编造证书、徽章、检测结果或专家背书。
- 使用绝对化安全、医疗或功效声明。
- 用伪官方样式制造权威感。

## Optimization Notes

- 先确认用户最担心的问题，再选择对应证据。
- 定期复核认证有效期、服务范围和市场适用性。

## Future Ideas

- 建立证明材料索引与过期提醒。
- 按品类和市场沉淀高频信任障碍。
