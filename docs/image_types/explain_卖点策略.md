# 02 Explain｜卖点图

## Purpose

在短时间内解释产品为什么值得购买。

## User Psychology

用户需要快速确认产品能解决什么问题。清晰、可信的 3–5 个卖点比大段描述更容易被理解和记住。

## Good Examples

- 卖点来自已审核产品资料，使用短句表达。
- 图标、标题和正文形成稳定的信息层级。
- 每个卖点说明一个不同价值，不重复堆词。

## Bad Examples

- 使用未经证实的功效、绝对化承诺或虚假数据。
- 字号太小、句子太长、图标风格不统一。
- 只写形容词，没有对应的产品事实。

## Optimization Notes

- 优先保留能影响购买决策且有证据支持的卖点。
- 在移动端 3 秒扫读条件下检查理解速度。

## Future Ideas

- 建立品类卖点优先级库。
- 记录不同市场对功能、体验和价格价值的关注差异。
