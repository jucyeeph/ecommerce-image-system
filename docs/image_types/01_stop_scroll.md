# 01 Stop Scroll｜主图

## Purpose

在信息流和搜索结果中建立第一眼识别，让用户停留并点击。

## User Psychology

用户先判断“这是什么、是否与我有关、值不值得继续看”。产品必须在缩略图中成为唯一视觉中心，信息越少越容易完成判断。

## Good Examples

- 产品占画面 60%–80%，轮廓、标签和颜色清晰。
- 只保留产品名、Logo 和一个真实核心卖点。
- 背景与产品形成明确对比，移动端仍可辨识。

## Bad Examples

- 多个卖点、徽章、贴纸和道具争夺注意力。
- 产品太小，文字比产品更突出。
- AI 改变瓶型、Logo、标签或 SKU 颜色。

## Optimization Notes

- 优先测试产品比例、背景明暗和单一卖点，而不是不断增加元素。
- 以缩略图点击意愿和产品识别速度作为主要判断依据。

## Future Ideas
