# Color System｜色彩系统

## Purpose

建立统一、克制且不影响产品辨色的品牌色彩语言。

## Core Palette

| 名称 | 角色 | 使用原则 |
| --- | --- | --- |
| Champagne Gold | 高级点缀 | 用于细线、局部反射和少量装饰 |
| Warm Beige | 温暖背景 | 适合生活方式和柔和商业场景 |
| Ivory White | 清洁留白 | 适合主背景与信息安全区 |
| Soft Black | 深色对比 | 适合文字、高对比背景和轮廓强化 |
| Pearl White | 珠光层次 | 适合柔和高光与细腻材质表现 |

正式生产必须使用批准的品牌色值；未确认前不得自行补写 HEX、RGB 或 CMYK。

## Rules

- 背景服务于产品，不改变 SKU 颜色判断。
- 辅助色可跟随产品，但不得压过核心色彩体系。
- 同一套九图保持稳定的明度、饱和度和冷暖倾向。

## Avoid

- 高饱和荧光色
- 彩虹渐变
- 刺眼背景
- 导致色差的强色偏
- 缺乏层次的脏灰色
