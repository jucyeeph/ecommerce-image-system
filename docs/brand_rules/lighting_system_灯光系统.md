# Lighting System｜光影系统

## Purpose

用稳定的工作室光影解释产品轮廓、材质和价值感。

## Preferred Lighting

- 柔和工作室主光
- 侧逆光勾勒轮廓
- 电影感补光控制层次
- 局部高光解释玻璃、金属和珠光
- 与场景一致的真实阴影和反射

## Material Behavior

- 玻璃应保留透明度、边缘高光与真实折射。
- 金属应有受控反射，不出现随机镜像。
- 磨砂表面应柔和扩散，不变成镜面或塑料。
- 珠光、猫眼和亮片应保留微观层次，不使用均匀发光替代。

## Avoid

- 生硬直闪
- 无来源硬阴影
- 过曝高光
- 吞没轮廓的暗部
- 多个互相矛盾的光源方向
