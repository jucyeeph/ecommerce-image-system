# Composition System｜构图系统

## Purpose

保持产品中心、移动端可读和九图之间的视觉连续性。

## Core Rules

- 产品通常占画面 60%–80%；Aspire 等环境型图片可按任务降低比例。
- 居中构图优先，黄金分割用于需要方向感或场景关系的画面。
- 保留文字安全区和平台裁切安全区。
- Logo、标题、图标和装饰不得遮挡关键结构。
- 每张图只有一个视觉中心和一个主要营销目标。

## Information Hierarchy

```text
产品或结果
→ 核心信息
→ 支持信息
→ 装饰元素
```

## Avoid

- 产品过小
- 多个视觉中心
- 文字和图标堆叠
- 无意义道具
- 画面边缘存在易被裁切的关键信息
