# Typography System｜字体系统

## Purpose

建立高级、清楚、易读且适合多平台的文字层级。

## Typeface Roles

### Display Candidates

- Baskerville
- Didot
- Playfair Display

用于主标题和少量高价值感文案。

### Supporting Candidates

- Helvetica
- Inter
- Montserrat

用于副标题、卖点、SKU 和说明文字。

## Rules

- 一套九图固定一组主标题字体和一组辅助字体。
- 字号、行距、字重和对齐方式保持一致。
- 文字必须满足移动端可读性，短句优先。
- 找不到指定字体时，只能使用已批准的相近替代字体。

## Avoid

- 同一套图片频繁更换字体
- 过度花体和低可读性字形
- 过长全大写文案
- 文字遮挡产品、Logo 或关键细节
