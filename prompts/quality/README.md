# 05 Quality｜细节图 Prompt 规范

## Objective

通过真实材质、工艺和结构细节证明品质。

## Required Inputs

- 产品高清图或微距参考
- 材质说明
- 关键工艺与结构信息
- 必须保留的细节清单

## Visual Requirements

- 使用微距或局部特写解释品质。
- 玻璃、金属、磨砂、珠光等材质反馈真实。
- 高光、阴影和反射符合物理逻辑。
- 关键结构清晰，不被文字或装饰遮挡。

## Forbidden

- 擅自增加不存在的纹理、结构或材质
- 模糊细节
- 错误反射和塑料感
- 通过过度锐化制造虚假质感

## Acceptance Checklist

- [ ] 关键细节与产品实物一致。
- [ ] 材质可被准确辨识。
- [ ] 微距没有改变结构和比例。
- [ ] 画面明确支持品质判断。
