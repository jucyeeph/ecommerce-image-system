# 07 Aspire｜向往图 Prompt 规范

## Objective

通过相关的理想环境和身份表达，建立用户认同。

## Required Inputs

- 目标用户画像
- 与品类相关的理想场景
- 产品图片
- 品牌视觉关键词

## Visual Requirements

- 环境高级、克制，并与产品真实使用场景相关。
- 产品仍可识别，不被空间和装饰淹没。
- 场景体现目标用户想要的工作或生活状态。
- 光影、色彩和材质符合品牌规范。

## Forbidden

- 与品类无关的豪华元素
- 复制其他品牌门店或可识别资产
- 让环境取代产品成为唯一主角
- 不符合目标市场的刻板化人物和场景

## Acceptance Checklist

- [ ] 场景与目标用户和品类相关。
- [ ] 产品在环境中自然且可识别。
- [ ] 画面表达身份认同而非单纯炫耀。
- [ ] 不含竞品品牌资产。
