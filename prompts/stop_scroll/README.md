# 01 Stop Scroll｜主图 Prompt 规范

## Objective

让用户在信息流中停下来，并快速识别产品与核心价值。

## Required Inputs

- 产品主图及结构参考
- 官方 Logo
- 产品名称
- 一个已审核的核心卖点
- 品牌视觉规范

## Visual Requirements

- 产品占画面约 60%–80%，是唯一视觉中心。
- 构图简洁、对比明确，适合移动端缩略图。
- Logo、产品名称和核心卖点清晰但不抢夺主体。
- 产品形状、标签、颜色和包装必须与素材一致。

## Forbidden

- 多卖点堆叠
- 无关道具抢夺注意力
- 夸张促销贴纸或廉价广告字
- AI 重绘 Logo、标签或产品结构

## Acceptance Checklist

- [ ] 一眼能识别产品。
- [ ] 核心卖点在 3 秒内可读懂。
- [ ] 缩略图状态下主体和 Logo 仍清晰。
- [ ] 产品事实与输入素材一致。
