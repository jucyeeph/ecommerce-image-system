# 06 Use｜使用图 Prompt 规范

## Objective

展示正确、自然的使用过程，证明产品易用、顺手。

## Required Inputs

- 正确使用步骤
- 产品尺寸与握持参考
- 使用者或场景要求
- 安全与禁用动作说明

## Visual Requirements

- 动作、手势、握持和产品朝向符合现实。
- 场景与目标用户及品类相关。
- 产品在使用过程中清晰可见。
- 人物、宠物和工具比例自然。

## Forbidden

- 错误或危险的操作方式
- 畸形手部、错误握持或悬浮产品
- 与产品能力无关的场景
- 擅自增加配件或改变产品尺寸

## Acceptance Checklist

- [ ] 使用动作正确且可复现。
- [ ] 产品比例和方向准确。
- [ ] 场景能够证明易用性。
- [ ] 不含安全风险或误导性操作。
