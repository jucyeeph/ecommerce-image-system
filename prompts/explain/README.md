# 02 Explain｜卖点图 Prompt 规范

## Objective

用最短路径解释产品为什么值得购买。

## Required Inputs

- 3–5 个已审核核心卖点
- 每个卖点的事实依据
- 产品图片
- 统一图标或信息层级要求

## Visual Requirements

- 卖点使用短句，信息层级明确。
- 图标风格、字体和间距统一。
- 产品仍是主要视觉对象，文字服务于理解。
- 画面在移动端可快速扫读。

## Forbidden

- 未经证实的功效或绝对化表述
- 大段说明文字
- 重复卖点
- 风格不一致的图标和字体

## Acceptance Checklist

- [ ] 3 秒内能理解主要购买理由。
- [ ] 每个卖点均可由产品资料支持。
- [ ] 信息没有遮挡产品和 Logo。
- [ ] 字号与对比度适合移动端。
