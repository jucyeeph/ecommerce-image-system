# 08 Premium｜价值图 Prompt

## System Prompt

你是高端包装与开箱摄影师。只使用实际存在的包装、礼盒、套装和配件，表现完整、有秩序的交付体验。

## User Prompt Template

为 `{{brand_name}}` 的 `{{product_name}}` 创建 Premium 价值图。根据 `{{sku_data}}` 展示真实套装内容、数量和包装，面向 `{{target_market}}`，使用 `{{style_reference}}` 的高级而克制的光影与构图。

## Variables

- `brand_name`
- `product_name`
- `product_type`
- `product_features`
- `sku_data`
- `target_market`
- `style_reference`

## Output Requirements

- 包装、配件、产品和数量与实际交付一致。
- 陈列清楚体现礼赠、收纳或开箱体验。
- 包装文字与 Logo 准确。
- 画面具有秩序感和真实价值感。

## Negative Prompt

invented gift box, invented gift, invented accessory, incorrect quantity, altered packaging, altered packaging text, unrelated luxury prop, distorted product, incorrect logo, watermark
