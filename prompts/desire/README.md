# 04 Desire｜效果图 Prompt 规范

## Objective

展示可信的最终效果，让用户产生拥有欲。

## Required Inputs

- 产品图片
- 经验证的使用结果
- 适用品类与使用部位
- 效果参考及限制条件

## Visual Requirements

- 最终效果清楚、自然并符合产品能力。
- 产品或其使用结果仍是画面重点。
- 人体、宠物或物体结构真实。
- 效果图与品牌视觉语言一致。

## Forbidden

- 夸大或伪造使用前后差异
- 不可能的颜色、质感或功能结果
- 畸形手部、宠物或使用对象
- 用无关奢华场景代替效果证明

## Acceptance Checklist

- [ ] 效果可由产品资料支持。
- [ ] 使用对象和结果符合现实。
- [ ] 没有暗示未经批准的功效。
- [ ] 用户能明确看到产品带来的结果。
