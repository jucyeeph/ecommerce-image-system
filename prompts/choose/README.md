# 03 Choose｜SKU 图 Prompt 规范

## Objective

准确展示可选 SKU，降低用户的选择成本。

## Required Inputs

- 完整 SKU 清单
- 官方色号、型号、尺寸或套餐名称
- 对应产品与色彩参考
- SKU 排序规则

## Visual Requirements

- SKU 排列整齐，标签与实物一一对应。
- 色卡、型号、尺寸或套餐差异易于比较。
- 使用中性且不影响辨色的背景。
- 保持统一比例、视角和命名格式。

## Forbidden

- 生成不存在的 SKU
- 改变颜色、型号名称或套餐内容
- 标签错位
- 用强色偏光影影响产品辨色

## Acceptance Checklist

- [ ] SKU 数量与输入清单一致。
- [ ] 名称、顺序和产品一一对应。
- [ ] 颜色与官方参考一致。
- [ ] 用户无需猜测差异即可完成选择。
