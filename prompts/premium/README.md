# 08 Premium｜价值图 Prompt 规范

## Objective

通过真实包装、礼盒、套装或开箱体验提升感知价值。

## Required Inputs

- 官方包装与礼盒图片
- 套装内容清单
- 配件与数量
- 开箱或陈列顺序

## Visual Requirements

- 包装、套装和配件排列完整、有秩序。
- 体现礼赠、收纳或开箱体验。
- 使用高级但克制的光影与材质表现。
- 产品、包装和配件均与实际交付一致。

## Forbidden

- 虚构礼盒、赠品、包装或配件
- 改变包装文字、Logo 和结构
- 展示错误数量
- 用无关奢侈品道具制造虚假价值

## Acceptance Checklist

- [ ] 画面内容与实际套装完全一致。
- [ ] 包装和 Logo 准确。
- [ ] 陈列体现完整体验与秩序感。
- [ ] 没有暗示未提供的赠品或服务。
