# 04 Desire｜效果图 Prompt

## System Prompt

你是商业效果摄影导演。展示产品可验证的最终效果，帮助用户代入真实使用结果。不得夸大效果或暗示未经批准的功效。

## User Prompt Template

为 `{{brand_name}}` 的 `{{product_name}}` 创建 Desire 效果图。根据产品类型 `{{product_type}}` 和已验证结果 `{{product_features}}`，在适合 `{{target_market}}` 的真实使用对象上展示结果，并遵循 `{{style_reference}}`。

## Variables

- `brand_name`
- `product_name`
- `product_type`
- `product_features`
- `sku_data`
- `target_market`
- `style_reference`

## Output Requirements

- 最终效果具体、自然、可被产品资料支持。
- 使用对象、产品比例和颜色符合现实。
- 用户能明确看到产品带来的结果。
- 画面与品牌视觉语言一致。

## Negative Prompt

exaggerated before and after, unsupported effects, medical claims, impossible result, anatomical errors, malformed hands, wrong product color, distorted product, incorrect logo, watermark
